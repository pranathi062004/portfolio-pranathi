# Personal Portfolio - Pranathi Shenoy

## Overview
This is a professional personal portfolio built for the Gesix Solutions Sprint 01 Challenge. 
It serves as a digital identity for Pranathi Shenoy, showcasing skills, projects, and professional background.

## Live Demo
https://pranathi-shenoy.vercel.app/

## Tech Stack
- **Framework**: React.js 18+ (via Vite)
- **Styling**: Vanilla CSS (CSS Variables, Flexbox, Grid)
- **Design**: Dark Mode / High Contrast / Minimalist / Glassmorphism
- **Performance**: Optimized for Lighthouse 90+ Scores

## Project Structure
```
src/
├── components/   # Reusable UI components
├── sections/     # Main page sections
├── styles/       # Global styles
├── data/         # Resume and project data
└── hooks/        # Custom React hooks
```

## Setup & Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/pranathi062004/portfolio-pranathi.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Key Features
- Fully responsive, mobile‑first layout
- Dark theme with accessible color contrast
- Modular component‑based structure
- Semantic HTML for better accessibility
- Lighthouse Accessibility score above 90
- Lighthouse Performance score above 90

## Engineering Decisions
- Clear heading hierarchy for screen readers
- ARIA labels for better navigation support
- Lightweight styling without heavy UI libraries
- CSS Grid for flexible responsive layouts
